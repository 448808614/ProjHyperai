# 如何阅读本文档

文档按照流式布局，分为部-章-节结构。每一章前面带有数字，第一个数字代表属于某一部，后者则代表特定章节。流式布局意味着文档需要按顺序从头到尾阅读。*（跳着读也可以，但错过内容请重新阅读。事实上这么做效率不会有所提高。）*
文档不要求阅读每一部，但无论从哪一部开始看起，都必须先阅读标号为'0'的部分，即关于项目本身章节和当前章节。

文档的部分章节提供“在这之前你需要了解...”和“扩展阅读...”节，看不看无所谓，最好看一下。
